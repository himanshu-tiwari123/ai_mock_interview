 Built a full-stack AI-powered mock interview platform using Next.js 15, TypeScript, and Firebase. The app allows users to practice job interviews with a real-time voice AI agent powered by Vapi and ElevenLabs TTS, which conducts structured interviews based on the user's target role and tech stack. After each session, interview transcripts are automatically analyzed by Google Gemini 2.5 Pro using the Vercel AI SDK to generate detailed feedback : scoring candidates on Communication, Technical Knowledge, Problem Solving, Cultural Fit, and Confidence. Authentication is handled via Firebase with session-based cookies (httpOnly, secure). The platform supports two flows: generating a new interview via a Vapi workflow, and taking a pre-existing interview from the community.
